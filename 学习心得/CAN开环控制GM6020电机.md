# 一、GM6020 电机 CAN 控制核心运行逻辑
* 初始化：配置 CAN（1Mbps 波特率、过滤电机 ID）+ 设定电机参数（ID / 控制模式 / 安全阈值）+ 通信自检。

* 指令下发：按 GM6020 协议封装指令帧，≥100Hz 均匀下发，发送失败有限重试。
* 反馈处理：监听 / 解析电机反馈帧（电流 / 转速 / 故障码等），校验后缓存数据。
* 闭环 + 异常：PID 调节指令实现闭环控制；监测故障 / 通信超时 / 数据异常，异常时立即发零指令，严重则急停。
* 停止：正常停止线性降指令至零；紧急停止立即发零指令并禁用主动指令。



# 二、标准 CAN 库核心调用逻辑
* 调用顺序：上电仅初始化一次（CAN_Init→过滤→启动）；控制周期内封装→发送→检状态；接收分中断（读 / 缓存）、轮询（≥500Hz 查收）。
时序 / 优先级：下发≥100Hz、轮询接收≥500Hz；中断仅轻量操作，主循环优先异常处理→指令下发→解析反馈；多任务加互斥锁。
* 容错规则：发送失败最多重试 3 次；接收超时 / 接口报错，先发零指令，再记录故障 / 尝试恢复。
